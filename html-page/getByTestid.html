Playwright getByTestId() Demo with Custom Attribute

Each button has both data-testid (default) and data-pw (custom). You can configure Playwright to use data-pw via selectors.setTestIdAttribute('data-pw').

1) Two buttons with the same text

Text is the same, but test ids are unique (save-top, save-bottom) in both data-testid and data-pw.

2) Text may change, test id stays stable

If text changes (e.g. "Directions"), both test ids (data-testid and data-pw) remain stable.

3) Icon button with no visible text

Icon-only button → No label/text. Test ids (settings) are the only reliable way to locate it.
